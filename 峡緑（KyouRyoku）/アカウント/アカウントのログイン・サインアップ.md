# アカウントのログイン・サインアップ
峡緑アカウントへのログイン・サインアップはすべて同じ手順で行われます。

まずは[ログインページ](https://kyouryoku.net/account/login)にアクセスしてください。峡緑アカウントへのログインは他のプロバイダのアカウントを用いたOAuth形式で行われます。サポートされているアカウントのプロバイダは以下のとおりです。
- Google
- Microsoft
- Line
- X
- Github
上記のどのアカウントでもログインすることができます。自身の好きなプロバイダを選択してください。

ログインが完了すると自動的に元のページに飛ばされます。

また、初めてのログイン（サインアップ）の場合アカウント情報が自動的に作成されます。この自動作成されるアカウント情報はプロバイダから提供されるアカウント名やプロフィール画像などに基づいて作成されます。もし、アカウントの情報を変更したい場合は[アカウントの管理](/峡緑（KyouRyoku）/アカウント/アカウントの管理.md)を参考にして変更してください。
