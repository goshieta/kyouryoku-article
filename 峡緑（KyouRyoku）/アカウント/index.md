# 峡緑アカウントについて
峡緑アカウントを使用すると、峡緑の運営する各種サービスのうち、ログインが必要な機能にア
クセスすることができるようになります。

アカウントのヘルプは以下の通りです。
- [アカウントのログイン・サインアップ](峡緑（KyouRyoku）/アカウント/アカウントのログイン・サインアップ.md)
- [アカウントの管理](/峡緑（KyouRyoku）/アカウント/アカウントの管理.md)

また、峡緑アカウントにログイン・サインアップすると、[利用規約](/峡緑（KyouRyoku）/利用規約.md)に同意したものとみなされます。
