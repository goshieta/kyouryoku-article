# 2(MhYg)について
![](assets/2mhyg_top.png)
[2(MhYg)](https://2mhyg.kyouryoku.net)は[峡緑](https://kyouryoku.net)が提供するSNSサービスです。「自分史」とよばれる自分の経験を、臨場感あふれる表現で投稿することができます。
## 2(MhYg)の説明
2(MhYg)とは峡緑の立ち上げた、ブログとSNSの中間的なサービスです。

当サイトでは、「自分史」と呼ばれるこれまで経験してきたこと、感じてきたこと、あるいは考えてきたことなどの自分の歴史を投稿することができます。（主にそうだというだけで、日常の出来事などその他の投稿も歓迎します。）

当サイトは、「自身の経験を匿名で公開することができ、他の人にコンテンツとして楽しんでもらえる。また、集合知として自身の経験を他の人の行動に役立てることができる。」ことを目標として設計されており、そのために以下のような特徴を持っています。

- 自分史を音や全画面の画像などで、よりリアルに感じられるように表現できる
- 完全匿名という特徴を活かすため、アカウントがなくても投稿、閲覧することができる。
